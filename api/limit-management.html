Limit Management

Endpoint:https://uat-api.enza.cloud/TX/v1

Manage card limits and restrictions

Retrieve Card Limits

GET
https://uat-api.enza.cloud/TX/v1
/limits/retrieve/{card_Id}

Retrieve the list of limits and restrictions configured on a card/token.

Pre-requisite: The target CardID must exist.

Retrieve Card Limits › path Parameters

  • card_Idstring · required

    Required. Card ID (numeric string)

Retrieve Card Limits › Headers

  • x-api-keystring · required

    API key for authentication

Retrieve Card Limits › Responses

Card limits successfully retrieved

  • codestring · required
    Example: SUCCESS_LIMITS_RETRIEVED
  • paramsobject · required
GET /limits/retrieve/{card_Id}

Update Card Limit Usage

POST
https://uat-api.enza.cloud/TX/v1
/limits/update-usage

Enable or disable a specific card limit/restriction.

Pre-requisites:

  • The target CardID must exist
  • Use Card Limit Retrieval API to get the limitRid first (see GET /limits/retrieve/{card_Id})

Update Card Limit Usage › Headers

  • x-api-keystring · required

    API key for authentication

  • Content-Typestring · required

    Content type of the request body

    Default: application/json

Update Card Limit Usage › Request Body

  • cardIdstring

    Required. Card ID (numeric string)

    Example: 3657
  • limitRidstring

    Required. Limit reference ID (from Card Limit Retrieval API)

    Example: PurchaseAmountLimit
  • inUseboolean

    Required. Enable (true) or disable (false) the limit

    Example: false

Update Card Limit Usage › Responses

Limit usage successfully updated

  • codestring · required
    Example: SUCCESS_LIMIT_UPDATED
  • paramsobject · required
POST /limits/update-usage

Change Card Limit Value

POST
https://uat-api.enza.cloud/TX/v1
/limits/change-limit

Update the maximum value of a specific card limit/restriction.

Pre-requisites:

  • The target CardID must exist
  • Use Card Limit Retrieval API to get the limitRid first (see GET /limits/retrieve/{card_Id})

Change Card Limit Value › Headers

  • x-api-keystring · required

    API key for authentication

  • Content-Typestring · required

    Content type of the request body

    Default: application/json

Change Card Limit Value › Request Body

  • cardIdstring

    Required. Card ID (numeric string)

    Example: 3657
  • limitRidstring

    Required. Limit reference ID (from Card Limit Retrieval API)

    Example: PurchaseAmountLimitRI
  • maxValnumber · min: 0

    Required. Maximum limit value (cannot be negative)

    Example: 1500

Change Card Limit Value › Responses

Limit value successfully updated

  • codestring · required
    Example: SUCCESS_LIMIT_UPDATED
  • paramsobject · required
POST /limits/change-limit
