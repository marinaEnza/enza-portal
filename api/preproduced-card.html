Preproduced Card

Endpoint:https://uat-api.enza.cloud/TX/v1

Instant card issuance for new and existing customers

Link Prepaid Card for New Customer

POST
https://uat-api.enza.cloud/TX/v1
/preproduced/prepaid/new-customer

Link prepaid card for a new customer.

Link Prepaid Card for New Customer › Headers

  • x-api-keystring · required

    API key for authentication

  • Content-Typestring · required

    Content type of the request body

    Default: application/json

Link Prepaid Card for New Customer › Request Body

  • cardRefNumstring

    Required. Pre-produced card reference number

    Example: PREP001234
  • embossedNamestring

    Required. Name to emboss on card

    Example: JOHN DOE
  • lastNamestring

    Required. Customer last name

    Example: Doe
  • middleNamestring

    Optional. Customer middle name

    Example: Michael
  • firstNamestring

    Required. Customer first name

    Example: John
  • birthDatestring

    Required. Birth date (YYYY-MM-DD)

    Example: 1990-05-15
  • emailstring

    Required. Customer email

    Example: john.doe@example.com
  • mobilePhonestring

    Required. Customer mobile phone

    Example: +233501234567
  • streetTitlestring

    Required. Street address

    Example: 123 Main Street
  • zipstring

    Required. Postal/ZIP code

    Example: 00233
  • nationalIdstring

    Conditionally Required: National ID (required if passport and identityId not provided)

    Example: GHA-123456789-0
  • passportstring

    Conditionally Required: Passport number (required if nationalId and identityId not provided)

    Example: G1234567
  • identityIdstring

    Conditionally Required: Identity ID (required if nationalId and passport not provided)

    Example: ID-987654321

Link Prepaid Card for New Customer › Responses

Prepaid card successfully issued to new customer

  • codestring · required
    Example: SUCCESS_REQUEST_PROCESSED
POST /preproduced/prepaid/new-customer
