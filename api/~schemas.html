Customer

  • requestTypestring

    Required. Request type:

    • NewAccountNewCustomer: New customer
    • NewAccountExistingCustomer: Existing customer
  • cardRefNumstring

    Required. Card reference number

  • embossedNamestring

    Conditionally Required: Required for NewCustomer/ExistingCustomer scenarios. Optional for DummyCustomer. Name to be embossed on the card

  • accountNumstring

    Required. Account number to link to the card

  • firstNamestring

    Conditionally Required: Required when requestType is 'NewAccountNewCustomer'. Customer's first name

  • middleNamestring

    Optional. Customer's middle name

  • lastNamestring

    Conditionally Required: Required when requestType is 'NewAccountNewCustomer'. Customer's last name

  • nationalIdstring

    Conditionally Required: For NewCustomer/ExistingCustomer: one of nationalId, passport, or identityId is required. Optional for DummyCustomer. Customer's national ID

  • passportstring

    Conditionally Required: For NewCustomer/ExistingCustomer: one of nationalId, passport, or identityId is required. Optional for DummyCustomer. Customer's passport number

  • identityIdstring

    Conditionally Required: For NewCustomer/ExistingCustomer: one of nationalId, passport, or identityId is required. Optional for DummyCustomer. Customer's identity ID

  • birthDatestring · date

    Optional. Date of birth in YYYY-MM-DD format

    Example: 1990-01-01
  • emailstring · email

    Optional. Customer's email address

  • mobilePhonestring

    Optional. Customer's mobile phone number

  • streetTitlestring

    Optional. Customer's street address

  • zipstring

    Optional. Customer's postal code

PinSetting

  • panstring

    Required. The Primary Account Number (PAN) of the card

    Example: 1234567890123456
  • pinBlockstring

    Required. The encrypted PIN block containing the new PIN

    Example: ABC123DEF456

PinChanging

  • panstring

    Required. The Primary Account Number (PAN) of the card

    Example: 1234567890123456
  • oldPinBlockstring

    Required. The encrypted PIN block containing the current PIN for verification

    Example: OLD123PIN456
  • newPinBlockstring

    Required. The encrypted PIN block containing the new PIN to be set

    Example: NEW789PIN012

CreateCardStandard

  • requestTypestring · enum

    Required. Type of request - NewAccountNewCustomer, NewAccountExistingCustomer, or ExistingAccountExistingCustomer

    Enum values:
    NewAccountNewCustomer
    NewAccountExistingCustomer
    ExistingAccountExistingCustomer
    Example: NewAccountNewCustomer
  • accountNumstring

    Required. Account number to create or link to

    Example: ACC123456
  • cardRefNumstring

    Required. Card reference number (ExtRid) of pre-produced card

    Example: CARD789012
  • embossedNamestring

    Required. Name to emboss on the card

    Example: JOHN DOE
  • firstNamestring

    Conditionally Required: Required when requestType is 'NewAccountNewCustomer'. Customer's first name

    Example: John
  • lastNamestring

    Conditionally Required: Required when requestType is 'NewAccountNewCustomer'. Customer's last name

    Example: Doe
  • middleNamestring

    Optional. Customer's middle name

    Example: William
  • birthDatestring · date

    Optional. Birth date in YYYY-MM-DD format

    Example: 1990-01-15
  • nationalIdstring

    Conditionally Required: One of nationalId, passport, or identityId is required. Customer's national ID

    Example: 1234567890
  • passportstring

    Conditionally Required: One of nationalId, passport, or identityId is required. Customer's passport number

    Example: AB1234567
  • identityIdstring

    Conditionally Required: One of nationalId, passport, or identityId is required. External ID number provided by the bank (use this if customer is not working with nationalId or passport)

    Example: ID123456
  • emailstring · email

    Optional. Customer's email address

    Example: john.doe@example.com
  • mobilePhonestring

    Conditionally Required: Required when requestType is 'NewAccountNewCustomer'. Customer's mobile phone number

    Example: 1234567890
  • streetTitlestring

    Optional. Customer's street address

    Example: 123 Main Street
  • zipstring

    Optional. Customer's postal code

    Example: 12345
  • notificationStatusstring · enum

    Optional. Notification preference - enabled or disabled

    Enum values:
    enabled
    disabled
    Example: enabled
  • autoActivateboolean

    Optional. Set to true to automatically activate the card after creation (default: false)

    Example: false
    Default: false
